## Format de génération recommandé

Pour **Midjourney** :
```
[prompt] --ar 16:9 --style raw --v 6 --q 2
```

Pour **DALL-E** :
```
Utiliser le prompt directement
```

Pour **Stable Diffusion** :
```
[prompt], high quality, educational illustration, clean design
```

---

## Notes d'utilisation

1. **Consentement** : Vérifiez les droits des images générées selon la plateforme utilisée
2. **Adaptation** : Ajustez les prompts selon le format de sortie souhaité (bannière, icône, illustration)
3. **Cohérence** : Maintenez la palette violet (#667eea) / purple (#764ba2) / beige (#F8F8F2) sur toutes les images
4. **Logo** : Mentionnez toujours "ZenClass logo" pour maintenir le branding
5. **Style** : Demandez le même style d'illustration plate moderne sur toutes les images pour une cohérence visuelle
